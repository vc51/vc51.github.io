🍀 关于本站

最新在学 python 语言和网络工程这方面的知识，就想着找一块地方来写写笔记、巩固学习的知识，顺便写写生活等，毕竟时光匆匆总是要留下点东西的，尝试过了几个笔记 App。onenote 用着还不错，但是无可奈何同步是个问题；有道体验也不错,但是 markdown 非会员不能上传图片；其他的笔记也有不错的。但是考虑到后期导出、能否长期运营下去等问题，还是决定把这些文字放到自己搭建的网站上，可控性高，能掌握在自己手中。
如果您也想拥有自己的博客，记录学习过程及生活点滴；那么就来看一看我是怎么搭建的吧：
本站使用了 Hexo 的 Fluid 主题部署在 Github 上，在这之前使用google服务器 + 宝塔面板 + wordpress 搭建了一个网址导航网站，因为这是收费的，觉得成本太高，所以搭建主机的博客就想找一些开源的项目来搭建。除了用 wordpress 这种不用写代码就能搭建的项目之外也没有太多的经验，从 20 点开始动手搭建，到 21:00 分基本完毕。搭建过程中也碰到了许多问题，几乎都是因为自己没仔细看教程文档造成的。

我的域名是在阿里云以 12 元/年购买的，经过一番对比挑选最后选择了阿里云。快戳这里领取优惠，抢注你的专属域名吧！
